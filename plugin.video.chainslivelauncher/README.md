# Chains Live Player

Minimal Kodi video addon that launches **Chains Live** (`plugin.video.chainslive`) main menu.

## Requirements

- Kodi 21+ (Python 3)
- [Chains Live](https://github.com/TheRedWizard/plugin.video.chainslive) installed from the Chains repository

## Install

1. Install **plugin.video.chainslive** (Chains Live).
2. Install **plugin.video.chainsliveplayer** (zip or repo).
3. Open **Video add-ons → Chains Live Player**.

Chains Live’s source list (Favorites, community files, configured M3U/XML sources) opens immediately — the same view a wizard would open with `plugin://plugin.video.chainslive/`.
